Github and Forum Links

TripleA is an open source project hosted on Github. Community maps are hosted by the triplea-maps org, also on Github

AxisAndAllies.org

AxisAndAllies.org is a great resource for TripleA and Axis and Allies players. It has vast forums, alternate rules, and website.
