Github and Forum Links

  • The Source Forge Discussion Forums are where questions about the engine, development, and map-making can be asked.
  • The TripleA engine source code can be found on Github
  • The vast majority of TripleA maps can be found in the Github map repos

TripleA War Club

The TripleA War Club is a website and forums dedicated to playing TripleA. There is a ladder and and a lobby for live games that can be accessed by clicking "Play Online" on the first TripleA screen. Users may also play by email or forum (PBEM/PBF)

AxisAndAllies.org

AxisAndAllies.org is a great resource for TripleA and Axis and Allies players. It has vast forums, alternate rules, and website. TripleA players may also play by forum using the A&A.org forums.

Developers

Looking to help out? Here are some resources to get you started.

TripleA Development Forum

The dev forums are a place for devs and map-makers alike to gather and discuss new features, maps, and anything going on that doesn't deal with an actual game going on.

Github Issues

Please give detailed problem reports or feature requests under issues on Github, where a dev will address your issue.
