Windows Installation

You may need to download and install Java, if you do not already have it.

(The latest version is {{ site.triplea-version }}.)..

Run the installer, and you can now run TripleA from the start menu.

If you already have Java 8 installed, then you can download the file called triplea_{{ site.triplea-version }}_windows_installer.exe

Run the installer, and you will be able to launch TripleA from your start menu.

Mac OS X Installation

Download the latest mac version of TripleA, which will be called triplea_{{ site.triplea-version }}_mac.dmg.

Double click on the DMG file, and drag TripleA.app to your applications folder. Double click on TripleA.app to run. If you do not have Java already installed, Mac OS X will promt you to download it.

If you get a "TripleA is damaged and cannot be opened" waning, follow these steps:

Apple menu > System Preferences > Security & Privacy > General tab under the header "Allow applications downloaded from:"

Change "Allow Applications Downloaded From:" to "Anywhere"

Your setting will reset to "Mac App Store and identified developers" to secure your computer from malware in 30 days, so you must repeat this each time you update TripleA.

Linux Installation

You must first install Java 8 or later. To test if you have Java installed, run this command from the command line: java -version.

You can install and download Java from here.

Download the file triplea_{{ site.triplea-version }}_all_platforms.zip.

Unzip it to any directory and execute the command ./triplea_unix.sh.
